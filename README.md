# 🕒 Digital Clock JS

A simple and elegant **real-time digital clock** built using JavaScript.
This project displays the user's **local time** dynamically and updates every second.

---

🚀 Live Demo

🌐 https://liveclock.onrender.com

---

## 🚀 Features

* ⏱️ Real-time clock
* 🌍 Displays local system time
* 🔄 Auto updates every second
* 🎯 Clean and minimal UI
* 💡 Beginner-friendly JavaScript logic

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (Date & Time API)

## 🎮 How It Works

* Uses JavaScript `Date` object
* Fetches current time from system
* Updates UI every second using `setInterval()`
* Displays time in **HH:MM:SS AM/PM** format
